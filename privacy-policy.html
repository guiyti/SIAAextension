Política de Privacidade

Dados Coletados

Esta extensão NÃO coleta dados pessoais dos usuários.

Processamento de Dados

A extensão processa apenas dados acadêmicos públicos do sistema SIAA para gerar relatórios CSV.

Armazenamento

Dados são armazenados temporariamente apenas no navegador do usuário para download de arquivos.

Compartilhamento

Nenhum dado é compartilhado com terceiros ou enviado para servidores externos.
