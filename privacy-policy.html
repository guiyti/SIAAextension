🔒 Política de Privacidade

SIAA Data Extractor v1.1.0
Última atualização: Dezembro 2024
Interface Responsiva e Headers Fixos

📋 Resumo

Esta extensão NÃO coleta, transmite ou compartilha dados pessoais. Todos os dados processados permanecem exclusivamente no seu navegador.

🔍 Dados Processados

A extensão processa apenas dados acadêmicos públicos do sistema SIAA:

  • Informações de disciplinas e ofertas acadêmicas
  • Dados de vagas e disponibilidade
  • Informações de campus e períodos
  • Nomes de professores e disciplinas

💾 Armazenamento Local

Os dados são armazenados apenas no seu navegador através de:

  • chrome.storage.local: Quando usado como extensão
  • localStorage: Quando acessado como arquivo local
  • Configurações de interface: Larguras de colunas, ordem, visibilidade

🌐 Transmissão de Dados

NENHUM dado é enviado para servidores externos. A extensão apenas:

  • Acessa o sistema SIAA que você já está logado
  • Processa dados localmente no seu navegador
  • Gera arquivos CSV para download local

🔐 Permissões Utilizadas

  • activeTab: Acesso à aba atual do SIAA
  • storage: Salvamento de dados e configurações localmente
  • scripting: Execução de scripts para extração (CSP compliance)

📱 Funcionalidades de Interface

A nova interface responsiva (v1.1.0) processa localmente:

  • Detecção de dispositivo mobile/desktop
  • Cálculos de altura para headers fixos
  • Configurações de layout responsivo
  • Persistência de preferências do usuário

🚫 O Que NÃO Fazemos

  • ❌ Não coletamos dados pessoais
  • ❌ Não enviamos dados para servidores
  • ❌ Não rastreamos atividade do usuário
  • ❌ Não acessamos dados fora do SIAA
  • ❌ Não compartilhamos informações com terceiros

🔄 Atualizações

Esta política é atualizada quando há mudanças significativas na extensão. A versão atual corresponde ao SIAA Data Extractor v1.1.0.
